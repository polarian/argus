# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1, as its code of conduct.

By participating, you are expected to uphold it. Please report unacceptable behavior
to the maintainer ([@polarian](https://github.com/polarian)) or via a
[private report](https://github.com/polarian/argus/security/advisories/new).
